# AIアプリポータル

このリポジトリは、AIと作った各種アプリをまとめる静的ポータルです。

## 公開構成

- `index.html`: GitHub Pagesで最初に開くポータルページ
- `portal-assets/`: ポータル専用のタイトル画像とスクリーンショット
- `sumureco/`, `sumucopy/`, `quest-simulator/`, `sumusale/`, `sumucal/`, `CATSAMURAI40/`: 各アプリ本体

GitHub Pagesでは、リポジトリ直下を公開対象にすると `index.html` から各アプリへ相対リンクで移動できます。
